Para n >= 5:
   q_n = aproximadamente 37%;
sendo 'n' o número de pessoas no Amigo Secreto;

O caso é um desarranjo, no qual nenhum dos valores está no seu lugar original. Em outras palavras p[0] != 0, p[1] != 1 e assim por diante.
A fórmula geral do desarranjo é a seguinte:
   Dn = n! * (1 - 1/1! + 1/2! - 1/3! + 1/4! - ... + (-1)^n * 1/n!)
Para n = 5, por exemplo:
   Dn = 5! * (1 - 1/1! + 1/2! - 1/3! + 1/4! - 1/5!) = 44
Sendo 44 o número de permutações válidas. Considerando que o total de permutações para n = 5 é 5! = 120:
   44/120 = aprox. 37%
E a partir de 5, o valor se mantém próx. de 37%.

A explicação parte das possibilidades para cada "casa" da permutação. Usando novamente n = 5:
Para p[0] temos 4 possibilidades, já que o próprio zero não pode entrar.
Se preenchermos p[0] com 1, para p[1] ficam duas possibilidades:
   Se preenchermos com 0 (ou seja, 0 e 1 trocam de lugar), restam as casas 3, 4 e 5 para permutar os números 3, 4 e 5. O problema se torna o mesmo que permutar para n = 3;
   Se preenchermos com um número != 0 (1 entra no lugar do 0, mas 0 não preenche o lugar do 1), o problema se torna o mesmo que permutar para n = 4;
Daí surge:
   Dn = (n - 1)*(d(n-1) + d(n-2))
A partir desse pensamento e utilizando outros princípios matemáticos, surge a fórmula geral.
